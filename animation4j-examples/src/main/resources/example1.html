This example shows a simple animation, which alternate directions and using an ease-in interpolator function for x and for the image angle.

animation {
    alternateDirection
    keyframe x, 0, 150, ease-in
    keyframe x, 1000, 350
    keyframe angle, 0, 0, ease-in
    keyframe angle, 1000, PI/2
}

Press Enter to restart the animation.
