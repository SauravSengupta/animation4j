Animation4j provides utility classes to work create animations in Java, based on time lines and interpolators.

It is a maven project and it is based on the next modules: 

	animation4j-core 

		This module contains all animation4j core classes and utilities.

	animation4j-componentsengine

		This module provides implementations of animation4j core module to be used 
		with components engine project.

	animation4j-slick

		If you are already using Slick2D, this module provides you with specific 
		classes to easier your work with animation4j.

	animation4j-examples

		This module provides examples of use of animation4j-core.
